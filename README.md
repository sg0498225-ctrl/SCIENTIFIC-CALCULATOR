Scientific Calculator (Python – Tkinter)

This project is a Scientific Calculator built using Python and Tkinter. It provides a clean graphical interface to perform basic arithmetic operations as well as scientific calculations like trigonometric functions, logarithms, powers, and square roots.

Features

User-friendly GUI built with Tkinter

Basic operations: +, −, ×, ÷

Scientific functions: sin, cos, tan, log, ln, √, power

Clear and delete options

Responsive button layout


Tech Stack

Python 3

Tkinter (GUI)

math module


How to Run

1. Install Python on your system


2. Run the script:

python scientific_calculator.py


3. The calculator UI will open automatically.
